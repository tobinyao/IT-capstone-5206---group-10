## Workflow: GitFlow + Kanban

- **Feature branches** created for each major component (e.g., `feature/map-interface`, `feature/data-import`)
- Code merged to `develop` branch after local testing and peer review
- **WIP limit** of 2 tasks per person to maintain steady workflow
- **Kanban columns:** Todo → In Progress → Review → Done
